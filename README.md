# BOLD 軽音 メンバーサイト

Next.js 16 + TypeScript で構築されたメンバー専用コミュニティサイトです。

## 主な機能

- **会員認証**: NextAuth.js v5（Google OAuth + パスワード認証）
- **投稿機能**: YouTube動画埋め込み、画像アップロード対応
- **スケジュール調整**: 複数候補日の設定と投票機能
- **リアルタイムチャット**: メッセージの即時配信
- **ユーザー管理**: 管理者による権限管理

## 技術スタック

- **フレームワーク**: Next.js 16 (App Router)
- **データベース**: Prisma + PostgreSQL
- **認証**: NextAuth.js v5
- **デプロイ**: Vercel
